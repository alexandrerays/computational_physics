E estou enviando um pequeno programa em java para exibir a onda do arquivo .raw e para converte-lo em WAVE

Exibe em uma janela gráfica a onda utilizando um período de mais ou menos 0,02 s:
java -jar RawReader <file>
OBS: Para fechar a janela pressione ESC
Converte o arquivo RAW em WAVE com 8 kHz de frequência de reprodução
java -jar RawReader -a0 <file>

Converte o arquivo RAW em WAVE com 44,1 kHz de frequência de reprodução
java -jar RawReader -a1 <file>

OBS: As duas saidas tem 8 bits/sample e Canal Unico.
